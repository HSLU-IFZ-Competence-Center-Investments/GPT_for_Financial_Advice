System requirements: Positional tracking technologies

On this page you can find the system requirements for all positional tracking technologies supported by Varjo headsets.

Varjo inside-out tracking

Varjo inside-out tracking is available for XR-4, and Varjo inside-out tracking (Beta) is available for XR-3.

With Varjo inside-out tracking, the positional tracking cameras on your headset scan the room to keep track of your position. No additional accessories are required.

Note that to use OpenVR applications (such as SteamVR apps) with XR-4, you need to install SteamVR software.

With XR-3, OpenVR applications can only be used with .

SteamVR™ Tracking

SteamVR™ Tracking is available for XR-4 base station tracking versions, XR-3, VR-3, Aero, VR-2 Pro, VR-2, XR-1 Developer Edition, and VR-1.

When using SteamVR™ Tracking, SteamVR™ base stations are used to track your headset and controllers. You can use either SteamVR™ 2.0 or 1.0 base stations for tracking.

Your base stations and accessories must be compatible with each other as described below:

VIVE Trackers (blue)
Valve Index “Knuckles” Controllers
Up to four base stations can be connected simultaneously.

Four base stations 2.0 are recommended if you plan to move around the room while using the headset.

Two base stations 2.0 or 1.0 are sufficient for sitting and stationary use.

Varjo headsets support OpenVR and OpenXR 1.0. SteamVR applications are compatible with Varjo headsets.

Varjo provides an API for providers of third-party tracking technologies to integrate their tracking solution with Varjo headsets. Read more about using third-party tracking methods.

Note that to use OpenVR applications (such as SteamVR apps), you may need to set up SteamVR™ Tracking for your headset. Refer to your tracking technology provider for details.
